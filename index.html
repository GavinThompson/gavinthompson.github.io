iamSick.ca

iamSick is a startup focused on helping patients find care whenever they need it. I worked with the iamSick team through 2013 as their sole developer and rebuilt an early prototype of the platform using Ruby on Rails with heavy integration of the Google Maps API.

During my time with the company, I implemented major features targeted at improving accessibility for non-english speaking patients in Canada. These features included the filtering of services by languages provided by each medical facility, and adding i18n to the app to translate the site to several languages spoken within Toronto.

Cool Shit, Good People -- The Platform

I wanted to create a platform which highlighted incredible people who are doing amazing things. With today's 24-hour news cycle, I found that good news often gets buried. Knowing that good news is out there, I wanted to build a platform which showcased individuals making the world a much better place.

Built using MeteorJS and ReactJS, the original intention was to allow users to post their own links. In April 2017, the project evolved and is now hidden behind a private URL acting solely as an endpoint for a seperate Chrome extension. Highlighted below, the extension is much more accesible to the general public and replaces the Facebook trending feed with links that I curate through the platform. Currently 900+ posts shared and counting!

Cool Shit, Good People -- Chrome Extension

As mentioned above, I built a Chrome extension! It takes all of the posts from the Cool Shit, Good People platform and replaces the Facebook "trending" section with these custom links of amazing people doing incredible things.

While I was very proud of the MeteorJS platform, I found this extension made the posts much more accessible. The extension displayed most recent, most popular, most visited, and random posts. It was built using a little bit of love and a whole lot of JavaScript wizardry.

Update - January 2019: Facebook removed their trending feed and this extension no longer works. Keeping this section and link to extension up for posterity.

One Minute Mandalas

Originally designed as a room-scale installation using projectors, this project was downsized to be more accessible to anyone with a computer or a smartphone. So far the feedback has been very positive! ðŸ˜Š

There's a line that keeps getting thrown around within the developer community: "build what you will use." In 2016, I had begun to experiment with meditation. During this period, I was inspired to build an installation that allowed participants to "release a thought" during meditation. Those thoughts are then converted through the one minute mandala algorithm, and a unique mandala is generated and drawn onto the screen.

One Minute Mandalas was built with P5js, a JavaScript library which reinterprets the Processing language for the web.

Yourem Ipsum

Several years ago lorem ipsum generators were incredibly popular. Ipsums like Bacon Ipsum, Cat Ipsum, and Samuel L Ipsum were all great examples of this fad. At the time, I thought there might be a need for copywriters and designers to have the ability to generate personal, customizable ipsums.

Yourem Ipsum was built with MeteorJS. Users upload a list of words and the applet returns several randomized ipsumized paragraphs.
